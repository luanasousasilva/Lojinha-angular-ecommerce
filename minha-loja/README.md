# 🛒 Lojinha Angular - E-commerce

Projeto de uma aplicação de e-commerce desenvolvida com Angular, com foco em boas práticas de desenvolvimento Front-End, organização de código e arquitetura baseada em componentes.

O objetivo do projeto é simular o fluxo básico de uma loja virtual, incluindo listagem de produtos, carrinho de compras e gerenciamento de estado da aplicação.

---

## 🚀 Tecnologias utilizadas

* Angular
* TypeScript
* RxJS
* NgRx (gerenciamento de estado)
* HTML5
* CSS3
* Node.js
* NPM

---

## 📂 Estrutura do projeto

```
src/
 ├── app/
 │   ├── components
 │   ├── pages
 │   ├── services
 │   └── store
 ├── assets
 └── environments
```

A aplicação segue uma arquitetura baseada em **componentes reutilizáveis**, separação de responsabilidades e organização modular.

---

## ⚙️ Como executar o projeto

### 1. Clonar o repositório

```
git clone https://github.com/seu-usuario/lojinha-angular-ecommerce.git
```

### 2. Acessar a pasta do projeto

```
cd lojinha-angular-ecommerce/minha-loja
```

### 3. Instalar dependências

```
npm install
```

### 4. Rodar o projeto

```
ng serve ou npm start
```

A aplicação estará disponível em:

```
http://localhost:4200
```
